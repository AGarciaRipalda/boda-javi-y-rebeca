# Boda Javi y Rebeca

Invitación web estática para la boda de Javi y Rebeca.

## Desarrollo

1. Instala dependencias con `npm install`.
2. Arranca el entorno local con `npm run dev`.
3. Genera la versión de producción con `npm run build`.

## Validación

- `npm run lint`
- `npm run typecheck`
- `npm run build`
